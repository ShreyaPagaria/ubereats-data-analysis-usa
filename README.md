# ubereats-data-analysis-usa
Exploratory Data Analysis &amp; Visualizations using Python <br>
Dataset: https://www.kaggle.com/datasets/ahmedshahriarsakib/uber-eats-usa-restaurants-menus <br> 
This project analyzes restaurant and menu data from the Uber Eats USA dataset, exploring trends in pricing, restaurant ratings, cuisines, menu structure, and geographic patterns across thousands of restaurants in the United States.

The goal is to uncover meaningful insights about food delivery patterns, restaurant behavior, and customer preferences using Python visualizations.

Example Visualizations

1. Top cuisines bar chart
2. Price distribution histogram
3. Price vs ratings boxplot
4. Menu category distribution
5. USA restaurant map
6. Average price per cuisine chart
